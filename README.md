# Practical-1: Student Portfolio Website

## 📝 Project Summary (What We Have Done)

In this practical, we developed a modern, responsive Single-Page Application (SPA) student portfolio using **React 19**, **Vite**, and **Vanilla CSS** with the following key implementations:

1. **Modular React Structure**: Divided the application into self-contained components (`NavBar`, `Header`, `About`, `Skills`, and `Footer`), each with co-located CSS files.
2. **Dynamic Accent Theme Switcher**: Added a React state-driven theme customizer in the navbar allowing users to switch global accents in real-time (Purple, Teal, Orange, Rose, Blue).
3. **Scroll-Spy & Smooth Navigation**: Integrated a dynamic scroll listener to highlight active navigation links automatically and enable smooth scroll offsets.
4. **Student Details & Profile**: Customized the app to display profile information for **Smit Sureja** (B.Tech Computer Science student at **Charusat University**).
5. **Interactive Skills Dashboard**: Grouped programming skills by category and rendered progress bars that color-adapt dynamically to the selected theme.
6. **Responsive Glassmorphism Styling**: Implemented modern visual details including blurred sticky headers, glow effects, hover transitions, and social channel integration.
